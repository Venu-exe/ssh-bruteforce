# SSH Brute Force Tool

> For educational purposes only. Use on systems you own or have permission to test.

## Requirements
```bash
pip install paramiko termcolor
```

## Usage
```bash
python3 sshburte.py
```

## Features
- Threaded brute force
- Color coded output
- Timeout handling
- Auto directory detection

## Tested On
- Metasploitable 2
- Kali Linux
